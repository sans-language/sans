Key Takeaways

  • Sans matches Rust on compute-heavy benchmarks (fib, loop_sum, array_ops) — both compile to optimized native code via LLVM
  • 30x faster than Python on recursive function calls
  • Beats Go by 40-75% on tight loops and array operations
  • JSON is a known weak spot — the C runtime JSON implementation needs optimization

Methodology

  • Each benchmark runs 10 times (1 warmup + 10 timed)
  • Wall-clock time measured via Python time.perf_counter()
  • Sans, Go, and Rust are compiled ahead of time with optimizations
  • Python uses CPython, Node.js uses V8
  • All programs produce identical output for correctness verification
