Key Takeaways

  • Sans matches Rust on compute-heavy benchmarks (fib, loop_sum, array_ops, mixed) — both compile to optimized native code via LLVM
  • 31x faster than Python on recursive function calls
  • Beats Go by 30-60% on tight loops, array operations, and mixed workloads
  • Beats Python and Go on JSON — 1.7x faster than Python, nearly matching Rust
  • 4x faster than Python on concurrent workloads using native threads and channels
  • 45k req/sec HTTP throughput — matches Rust and Go; all languages are I/O-bound at this concurrency level

Methodology

  • Each benchmark runs 10 times (1 warmup + 10 timed)
  • Wall-clock time measured via Python time.perf_counter()
  • Sans, Go, and Rust are compiled ahead of time with optimizations
  • Python uses CPython, Node.js uses V8
  • All programs produce identical output for correctness verification
